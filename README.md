# Blackjack Game (Python)

A text-based Blackjack game built in Python as **Milestone Project 2** of the 
[Python Bootcamp: From Zero to Hero](https://www.udemy.com/course/complete-python-bootcamp/).

## Features
- Object-Oriented design using `Card`, `Deck`, `Hand`, and `Chips`
- Ace value adjustment logic
- Betting system with chip management
- Input validation for user actions
- Dealer logic (hits until hand value reaches 17)

## How to Run
```bash
git clone https://github.com/codergallery/BlackJack-Game
cd Blackjack-game\Game\
python blackjack.py
